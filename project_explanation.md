# Project Explanation

## Project Title
Student Course Registration Database System

## Purpose
The purpose of this project is to model a simple university course registration system using relational database concepts.

## Main Entities
- Students
- Lecturers
- Courses
- Enrollments
- Audit Log

## Relationships
- One lecturer can teach many courses
- One student can enroll in many courses
- One course can have many students
- Enrollments resolves the many-to-many relationship between students and courses

## Database Concepts Used
- Primary Keys
- Foreign Keys
- Unique Constraints
- Check Constraints
- Joins
- Aggregation
- Procedure
- Function
- Trigger

## Learning Outcome
This project helped me practice SQL querying, schema design, and PL/SQL concepts that are useful in database development roles.
